# changing colors
#CC 20 specifies columns numbered 1-20 (does this mean that the buttons on left are column 0?)
#CC 21 specifies rows numbered 0-7
#CC 22 specifies color, colors are
# 0=? , 1=Red, 2=Yel, 3=Gr, 4=Cy, 5=Bl, 6=Mag, 7=OFF, 8=White, 9=O 10=Lime 11=Pink

#separating commands with what looks like 10 milliseconds
#+00.010 means relative time till accepting next command is 10 milliseconds


CC commands
20          Column coordinate for cell color change with CC 22 (starts from 0)
21          Row coordinate for cell color change with CC 22 (starts from 0)
22          Change the color of the cell with the provided column and row coordinates
            see color value table below, 11+: default color

23			Make current custom cell colors persistent as pattern (0-2)  (0=A, 1=A#, 2=B these are three storage spaces for colors)
24			Clear persisted custom cell colors pattern (0-2)  (0=A, 1=A#, 2=B these are three storage spaces for colors)

NRPN 37     0-14    Split Left Transpose Pitch (0-6: -7 to -1, 7: 0, 8-14: +1 to +7)
137    0-14    Split Right Transpose Pitch (0-6: -7 to -1, 7: 0, 8-14: +1 to +7)
138    0-14    Split Right Transpose Lights (0-6: -7 to -1, 7: 0, 8-14: +1 to +7)
